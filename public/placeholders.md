# Asset Placeholders

These files belong in `/public/` and need to be added before going live:

| File | Purpose | Notes |
|---|---|---|
| `workbook.pdf` | The 24-page companion workbook | Designed from `03_Companion_Workbook.md`. Web-optimized (< 10 MB) for fast download. |
| `og-image.jpg` | Open Graph card (1200 × 630) | Shown when the course link is shared on social. Use the hero headline in Revivery brand style. |
| `favicon.ico` | Browser tab icon | 32 × 32 Revivery mark |
| `fonts/ABCArizonaSans-Regular.woff2` | Primary font | License from ABC Dinamo |
| `fonts/ABCArizonaSans-RegularItalic.woff2` | Primary font italic | License from ABC Dinamo |
| `fonts/ABCArizonaSans-Bold.woff2` | Primary font bold | License from ABC Dinamo |

Until the font files are added, the app falls back to Inter / system sans-serif and renders correctly.
